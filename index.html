Rails Webapp Project

DriveSimple is a web based platform that facilitates the booking of driving lessons by connecting driving instructors directly with students. Instructors can create a profile, and publish their schedules and availabilities. On the other side, students can browse, compare, and review instructors before booking a lesson. DriveSimple streamlines the booking process by offeringlisting, scheduling, and payment all in one place.

  • Developed using Agile Design Principles, Ruby on Rails 4.2.2 + Boostrap 3
  • Deployed using Heroku

Example-Directed Program Synthesis

The primary objective of this project was to explore in details example-directed program synthesis concepts, mainly concentrating on the new method by Osera and Zdancewic. This includes a careful review of the theoretical foundations of program synthesis, both in general and specifically example-directed. This project also reproduces some of the test cases presented in their paper using MYTH and propose potential extensions and improvements to the existing work.

AI Project

An AI agent was developed for the game of Hus. The main goal of this project was to apply theories of artificial intelligence to practice, and create an AI agent that out performs other agents.The backbone of this agent is a standard MinMax search algorithm with Alpha-Beta pruning, in pseudo-randomized order. The main design tasks are to determine the maximum number of search depth within the memory and time constraint, and to optimize the search heuristic. These design tasks are mainly achieved through exhaustive simulation using different parameters, the details of which are discussed in the report.
